How to use:
	1) Put .wav files inside the Songs folder
	2) Run main.exe
	3) Click Sort by BPM
	4) On the terminal you will see the calculations 
	5) When the program is finished you will see a pop-up window 
	6) The playlist will be exported inside the folder as sorted_by_bpm.txt

Warning:
	Program runs slow so have patience. The bigger the wav file the more time it will take.
	There are a few small .wav files inside the Songs folder already so you can test the program.
	Filenames of .wav files have they bpm so you can check the accuracy of the sorted playlist.
